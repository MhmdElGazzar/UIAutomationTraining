XPath Beginner Practice

Exercise 1: Basic Element Selection

Find the XPath for the button with text "Click Me":

XPath: //button[@id='simple-button']

Alternative: //button[text()='Click Me']

Exercise 2: Element with Class

Find the XPath for the paragraph with class "special-text":

This is a special paragraph

This is a normal paragraph

XPath: //p[@class='special-text']

Exercise 3: Element with Multiple Attributes

Find the XPath for the input field with type "text" and name "username":

XPath: //input[@type='text' and @name='username']

Exercise 4: Element by Index

Find the XPath for the second list item:

  • First item
  • Second item
  • Third item

XPath: //li[2]

Alternative: //li[text()='Second item']

Exercise 5: Element by Text Content

Find the XPath for the link containing "documentation":

Home View documentation Contact

XPath: //a[contains(text(), 'documentation')]

Exercise 6: Element with Specific Parent

Find the XPath for the span inside the div with class "container":

Target text
Other text

XPath: //div[@class='container']/span

Exercise 7: First Element of Type

Find the XPath for the first paragraph in the document:

First paragraph

Second paragraph

Third paragraph

XPath: (//p)[1]

Exercise 8: Element with Specific Attribute Value

Find the XPath for the image with alt text "Logo":

XPath: //img[@alt='Logo']

Exercise 9: Last Element of Type

Find the XPath for the last div in the container:

First div
Middle div
Last div

XPath: //div[@class='wrapper']/div[last()]

Exercise 10: Element with Multiple Classes

Find the XPath for the element with both classes "btn" and "primary":

XPath: //button[@class='btn primary']
