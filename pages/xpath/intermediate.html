XPath Intermediate Practice

Exercise 1: Contains Function

Find the XPath for all links containing the word "product" in their href:

Product One About Us Product Two Contact

XPath: //a[contains(@href, 'product')]

Exercise 2: Multiple Conditions

Find the XPath for input fields that are required and of type "email":

XPath: //input[@type='email' and @required]

Exercise 3: Following Sibling

Find the XPath for paragraphs that come after a heading:

Section 1

Find this

And this

Not this

Section 2

Find this too

XPath: //h3/following-sibling::p

Exercise 4: Starts-with Function

Find the XPath for elements with IDs starting with "user-":

Profile
Settings
Admin

XPath: //*[starts-with(@id, 'user-')]

Exercise 5: Position Function

Find the XPath for every odd-numbered row in the table:

Row 1
Row 2
Row 3
Row 4

XPath: //tr[position() mod 2 = 1]

Exercise 6: Parent Axis

Find the XPath for the parent div of a span with text "Target":

Other
Target

XPath: //span[text()='Target']/parent::div

Exercise 7: Ancestor Axis

Find the XPath for all ancestor divs of a specific paragraph:

Find ancestors

XPath: //p[@id='target']/ancestor::div

Exercise 8: Preceding Sibling

Find the XPath for all list items before the one with class "current":

  • Item 1
  • Item 2
  • Current Item
  • Item 4

XPath: //li[@class='current']/preceding-sibling::li

Exercise 9: Text Contains and Normalize Space

Find the XPath for paragraphs containing the word "important" (case-insensitive):

This is Important text.

Regular text

Another IMPORTANT notice

XPath: //p[contains(translate(text(), 'IMPORTANT', 'important'), 'important')]

Exercise 10: Complex Attribute Selection

Find the XPath for inputs with a data-type attribute containing either "text" or "number":

XPath: //input[contains(@data-type, 'text') or contains(@data-type, 'number')]
