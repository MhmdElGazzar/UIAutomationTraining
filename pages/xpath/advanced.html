XPath Advanced Practice

Exercise 1: Complex Predicates with Functions

Find the XPath for divs that have exactly 3 child paragraphs, where the second paragraph contains "target":

First

This is the target text

Last

One

Two

XPath: //div[count(p)=3 and p[2][contains(text(), 'target')]]

Exercise 2: Dynamic Attribute Matching

Find the XPath for elements with a data-timestamp from today (dynamic date):

Today's content
Yesterday's content
Tomorrow's content

XPath: //*[substring(@data-timestamp, 1, 10) = substring(string(current-date()), 1, 10)]

Exercise 3: Complex Ancestor-Descendant

Find the XPath for spans that are descendants of a div with class "container" but not inside a paragraph:

Find this

Not this

Find this too

XPath: //div[@class='container']//span[not(ancestor::p)]

Exercise 4: Multiple Axis Navigation

Find the XPath for paragraphs that have a preceding h2 and a following button:

Section 1

Find this paragraph

Not this one

Section 2

Not a paragraph

XPath: //p[preceding::h2 and following::button[1]]

Exercise 5: Complex String Functions

Find the XPath for elements with text that starts with a number and ends with "items":

5 items
No items
10 items remaining
3 items

XPath: //*[matches(text(), '^\d+\s+items$')]

Exercise 6: Nested Conditions

Find the XPath for links that have both an image and text content, where the image has an alt attribute:

Link 1 Link 2 Link 3

XPath: //a[img[@alt] and normalize-space(text())]

Exercise 7: Complex Position Logic

Find the XPath for every third list item that also has a specific class:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5
  • Item 6

XPath: //li[@class='special' and (position() mod 3 = 0)]
